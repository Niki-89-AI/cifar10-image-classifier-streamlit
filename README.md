
# CIFAR-10 Image Classifier

This project implements a convolutional neural network (CNN) for image classification using the CIFAR-10 dataset. The model is enhanced with data augmentation, dropout regularization, and hyperparameter tuning.

## Features
- CNN-based image classification
- Data augmentation for improved generalization
- Dropout to reduce overfitting
- Hyperparameter tuning experiments
- Streamlit interface for real-time predictions

## How to Run

1. Install dependencies:
pip install -r requirements.txt


2. Run the app:
app.py


## Dataset
CIFAR-10 dataset (10 classes, 32x32 RGB images)
